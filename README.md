Slogan ("Kendi düzenini kur — sınırları genişlet!") + masa fotoğrafı + süreç daireleri +
**İletişime Geçin** butonu. (Üst bölüm ayrı projedir: `site-ust`.)

- Masaüstünde 1920px sabit tasarım orantılı ölçeklenir; **768px altında** mobil düzen devreye girer
  (süreç daireleri mobilde yatay kaydırmalı şerittir).
